UNIX / LINUX:
	1. The Unix operating system is a set of programs that act as a link between the computer and the user.
	   The computer programs that allocate the system resources and coordinate all the details of the computer's
	   internals is called the operating system or the kernel.
	2. Several people can use a Unix computer at the same time; hence Unix is called a multiuser system.
	3. A user can also run multiple programs at the same time; hence Unix is a multitasking environment.

19. grep [pattern] [file]:
	search a file like given pattern.
	grep -v -> print all line do not match pattern
	grep -n -> print the matched line.
	grep -l -> Prints only the names of files with matching lines (letter "l")  
	grep -i -> match either upper or lower.
	eg: ls -R /home/thamu/hi1/ | grep 2023_06_*
20. sort [file]:
	arrage the lines of text alphabetically or numerically.
	sort -n -> sort numerically.
	sort -r -> sort reverse order.
	sort -f -> sort upper and lower together.
	sort +x -> ignore first field when sorting.
	eg: ls -l | grep "Aug" | sort +4n

31.cut OPTION... [FILE]:
	it used to extract the text from given statement or file.
	it is like python slicing concept.
	-c or --characters: Specifies the characters to be extracted from each line. You can provide a comma-separated list or a range of characters.
	-f or --fields: Specifies the fields to be extracted from each line. Fields are separated by a delimiter (usually a tab or space).
	-d or --delimiter: Specifies the delimiter character used to separate fields. By default, it is set to a tab character.
	-b or --bytes: Specifies the bytes to be extracted from each line. You can provide a comma-separated list or a range of bytes.
	eg:
	while IFS= read -r line; do
    echo "$line" | cut -c 3 -> [(-c2,7)(-c2-7),cut -d $'\t' -f1-3]
	done

		--statement

	done

		--statement
	done
	
for ((i=1; i<=n; i++)); do
	--statement
done
